ClassicalSpinMC.AbstractObservables — Type
abstract type AbstractObservables

This is a struct to store Observables measured during the MonteCarlo run.

!!! note In order to measure observables other than the built-in Observables type or a subset of it, you may create a subtype of AbstractObservables such as MyObservables and define the following functions.

1.performMeasurements!(observables::MyObservables, lattice::T, energy::Float64) where {T<:Lattice}

2.performPostMeasurements!(observables::MyObservables, lattice::T, β::Float64) where {T<:Lattice}

Depending on the complexity of your type, you may also have to define

1.writeObservables!(fn:H5, obs::MyObservables)

2.readObservables

if the general function cannot handle those types.

ClassicalSpinMC.initSpinConfiguration! — Method
function initSpinConfiguration!(lattice::Lattice{D,N}, f::typeof(conicalUpdate), rng=Random.GLOBAL_RNG) where {D,N}

Initialize the spin configuration of a lattice using the conicalUpdate method.

Pre-initialize for conical updates

Since conicalUpdate forms a cone around a point on a unit sphere, it cannot be used to reliably initialize a spin configuration unless it was already initialized using a different method first. This method is provided only for consistency.
